# 🌸 FunsDiia Bot

Повнофункціональний Telegram-бот для генерації документів з адмін-панеллю, промо-кодами, рефералками та запуском на GitHub Actions.

---

## 🚀 Запуск на GitHub Actions

### 1. Secrets (Settings → Secrets → Actions)

| Секрет | Опис |
|--------|------|
| `TELEGRAM_BOT_TOKEN` | Токен від @BotFather |
| `ADMIN_IDS` | ID адмінів через кому: `123456789,987654321` |
| `BOT_USERNAME` | Username бота без @ |

### 2. Push в main → бот запускається автоматично

## 👑 Функції адмін-панелі

### 📊 Статистика
- Кількість користувачів (всього / активних / VIP)
- Замовлення (виконано / в черзі / відхилено)
- Фінансова статистика (дохід, баланс юзерів)
- Нові юзери та замовлення за 24 години

### 📋 Замовлення
- Перегляд з фільтрами (⏳ / ✅ / 🎉 / ❌)
- Підтвердження → автоматична відправка реквізитів
- Відхилення з поясненням причини
- Надсилання готових файлів (auto-forward до клієнта)

### 👥 Користувачі
- Список з бейджами (VIP / бан / новий / покупець)
- Пошук за @username, ID, або ім'ям
- Бан / розбан одним кліком
- Надання / зняття VIP-статусу
- Зміна балансу
- Написати клієнту з панелі

### 💰 Тарифи
- Увімк./вимк. тариф
- Редагування назви, ціни, емоджі
- Видалення тарифу
- Додавання нового тарифу (покроково)

### 🎟️ Промо-коди
- Створення промо-коду зі знижкою %
- Ліміт використань (або необмежений)
- Увімк./вимк. / видалення коду
- Юзер вводить код перед замовленням

### 📢 Розсилка
- Preview перед відправкою
- HTML-форматування
- Результат: успішно / помилок / заблоковано

### 💬 Відгуки
- Список з статусами (🟢 новий / 🔵 читано / 🟣 відповіли)
- Відповідь на відгук через панель

### ⚙️ Налаштування
- Технічне обслуговування (бот недоступний для юзерів)
- Вмикання/вимикання нових замовлень
- Зміна реквізитів оплати (картка / отримувач / посилання mono)
- Кастомний текст привітання

### 📜 Логи дій
- Останні 500 дій: реєстрації, замовлення, бани, балансові операції

---

## 📦 Функції для юзерів

- Каталог тарифів з актуальними цінами
- Замовлення (ПІБ → ДН → стать → фото)
- Промо-коди зі знижками
- Реферальна програма (бонус за кожного запрошеного)
- Профіль з балансом та статистикою
- Мої замовлення з статусами
- Зворотній зв'язок

---

## 🔧 Команди адміна

```
/admin      - відкрити адмін-панель
/stats      - швидка статистика
/broadcast  - розсилка
/ban <id>   - заблокувати
/unban <id> - розблокувати
/balance <id> <сума>  - нарахувати/списати баланс
```

## 📁 Файли даних

| Файл | Вміст |
